# Local Project Context

This folder stores internal memory and operational repository context in Markdown format.

## Files
- `architecture-context.md`: architecture vision and boundaries.
- `decision-log.md`: technical decisions (lightweight ADR).
- `domain-glossary.md`: domain ubiquitous language.
- `working-memory.md`: temporary state for active tasks.

## Usage Rule
- Update these files whenever a decision changes contracts, architecture, or key processes.
