# FE-02-03 — Add React Hook Form + Zod Resolvers

**Epic:** [FE-02 — Core Libraries Integration](../epics/FE-02-core-libraries.md)  
**Status:** `[x]` Done

## Description
Install React Hook Form and the Zod resolver integration. Create a sample form that validates on submit and displays field-level errors.

## Acceptance Criteria
- [x] `react-hook-form` and `@hookform/resolvers` are installed.
- [x] `zod` is installed as the primary schema validation library.
- [x] A sample form schema is defined with `z.object({...})` and used as the resolver.
- [x] Field-level errors are displayed next to each input on invalid submission.
- [x] Valid form submission logs the typed form values.
- [x] Form state types are inferred from the Zod schema (`z.infer<typeof schema>`).

## Technical Notes
- Use `useForm<z.infer<typeof schema>>({ resolver: zodResolver(schema) })`.
- Zod schemas should be in `lib/schemas/` for reuse across forms and API validation.

## Definition of Done
Sample form validates, shows errors, and submits typed data; no `any` types in form code.
