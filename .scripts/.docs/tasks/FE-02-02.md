# FE-02-02 — Configure TanStack Query with Base Client

**Epic:** [FE-02 — Core Libraries Integration](../epics/FE-02-core-libraries.md)  
**Status:** `[x]` Done

## Description
Install and configure TanStack Query with a shared `QueryClient` instance, global stale time defaults, and a global error handler for failed queries.

## Acceptance Criteria
- [x] `@tanstack/react-query` is installed.
- [x] A `QueryClient` is created with default options: `staleTime: 1000 * 60` (1 min), `retry: 1`.
- [x] `QueryClientProvider` wraps the app.
- [x] A global `onError` handler logs failed queries (without exposing sensitive data).
- [x] `@tanstack/react-query-devtools` is installed and rendered only in development.
- [x] A sample `useQuery` hook fetches from a mock endpoint and renders loading/data/error states.

## Technical Notes
- Create `src/lib/query-client.ts` to export the singleton `QueryClient`.
- Import it in `main.tsx` for the provider.

## Definition of Done
`QueryClientProvider` wraps the app; devtools appear in dev mode; sample query hook works.
