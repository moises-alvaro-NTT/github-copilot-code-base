# FE-06-01 — Set Up Storybook or Ladle

**Epic:** [FE-06 — Documentation and Playground](../epics/FE-06-documentation-playground.md)  
**Status:** `[x]` Done

## Description
Install and configure Storybook (or Ladle as a lighter alternative) with Tailwind and TypeScript support so component stories can be written and previewed.

## Acceptance Criteria
- [x] Storybook (or Ladle) is installed and configured for Vite + React + TypeScript.
- [x] Tailwind CSS is applied to story previews.
- [x] `npm run storybook` starts the playground on a local port without errors.
- [x] A sample story renders a `<Button>` component.
- [x] Storybook config is committed to source control.
- [x] Story files are co-located with components (`Button.stories.tsx` next to `Button.tsx`).

## Technical Notes
- For Storybook: use `@storybook/react-vite` builder.
- For Ladle: `@ladle/react` with Vite — lighter and faster for small projects.
- Configure the `preview.ts` to import global CSS (Tailwind directives).

## Definition of Done
`npm run storybook` starts cleanly; sample story renders `<Button>` with correct styles.
