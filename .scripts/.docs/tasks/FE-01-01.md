# FE-01-01 — Scaffold React + TypeScript Project

**Epic:** [FE-01 — Project Setup](../epics/FE-01-project-setup.md)  
**Status:** `[x]` Done

## Description
Initialize the frontend project using Vite (or Next.js) with the React + TypeScript template and enable strict mode.

## Acceptance Criteria
- [x] Project generated with `npm create vite@latest` (or equivalent) using the `react-ts` template.
- [x] TypeScript `strict: true` is set in `tsconfig.json`.
- [x] `noUnusedLocals` and `noUnusedParameters` are enabled.
- [x] `npm run dev` starts the development server without errors.
- [x] `npm run build` produces a production bundle without TypeScript errors.
- [x] `index.html` renders the root React app in a browser.

## Technical Notes
- Use Vite for speed unless Next.js SSR is explicitly required.
- Remove all default Vite boilerplate (App.css, logo, counter) after scaffolding.

## Definition of Done
Clean React+TypeScript project runs locally; strict mode enabled; boilerplate removed.
